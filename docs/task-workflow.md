# Task Workflow

1. Create project
   - `my-zk-ext project new my-first-project`
1. List projects to check created
   - `my-zk-ext project list`
1. Create task
   - `my-zk-ext task new`
1. List not-started task.
   - `my-zk-ext task list --status=not-started`
1. Mark task to doing
   - `my-zk-ext task mark Projects/my-first-project/taskfile.md doing`
